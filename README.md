This repo holds a static website hosting documents from John Hoyte using OpenSeadragon.
The first such document is his 1956 journal/scrapbook detailing the first trip of the "Cambridge Hannibalians"
to investigate the different passes across the Alps which Hannibal may have used in 218 BCE.

## Building DZI collection data file

You need to have Node.js installed. Then `cd util` and `npm install`. Then you can run `node collect.js`.
